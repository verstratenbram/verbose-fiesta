# verbose-fiesta (site)

Kleine repository met een simpele one-page profielsite voor Bram Verstraten.

Bestanden:

- [index.html](index.html) — de one-page website
- [styles.css](styles.css) — eenvoudige styling

Preview lokaal:

```bash
# vanaf de repo root
python3 -m http.server 8000
# open vervolgens http://localhost:8000 in je browser
```

Wil je inhoud of styling aanpassen, open `index.html` en `styles.css`.
